# Core Features

## Request Workflow

- Multi-request tabs with add/close behavior.
- Method selector with method-aware styling.
- URL editor with variable interpolation support.
- Send + Save actions always visible.

<div class="rp-shot">
  <img src="/screenshots/request-builder.png" alt="Request builder controls and tabs" />
</div>

## Collections and Saved Requests

- Collections stay visible in the left panel.
- Save flow lets you choose existing or create new collection.
- Collection items load full request config into active tab.

<div class="rp-shot">
  <img src="/screenshots/collections-panel.png" alt="Collections panel" />
</div>

## Response Experience

- Response metadata: status, time, size.
- Tabs for JSON, raw, headers, and test results.
- Copy actions and script logs in the same response context.

<div class="rp-shot">
  <img src="/screenshots/response-viewer-json.png" alt="Response JSON viewer" />
</div>

## History You Can Trust

- Each send creates a full snapshot.
- Request and response are inspectable in detail.
- Copy snapshot payloads for debugging/sharing.
