ReqPilot is a local-first REST API client for building requests, testing APIs, managing environments, and sharing collections.

> Data stays on your system. ReqPilot does not host your payloads in the cloud, and request execution remains under your control.

![ReqPilot application screenshot](./docs/public/reqpilot.png)

## Run Locally (NPX)

```bash
npx @toolstackhq/reqpilot
```

ReqPilot starts at `http://localhost:5489` in your browser.
